# Student Grade Calculator

A simple web-based calculator for entering student marks, calculating the average score, and assigning a grade.

## Features

- Add marks for multiple subjects
- Add or remove subject rows
- Calculate total marks and average percentage
- Assign grades based on the average score
- Show pass or improvement status
- Reset the calculator to start again
- Responsive layout for desktop and mobile screens

## Grade Scale

| Average Score | Grade |
| --- | --- |
| 90% and above | A |
| 80% to 89.99% | B |
| 70% to 79.99% | C |
| 60% to 69.99% | D |
| 40% to 59.99% | E |
| Below 40% | F |

## How to Use

1. Open `index.html` in a browser.
2. Enter subject names and marks from 0 to 100.
3. Use the plus button to add more subjects if needed.
4. Click **Calculate** to see the total, average, grade, and status.
5. Click **Reset** to clear the marks and start again.

## Project Files

- `index.html` contains the page structure.
- `styles.css` contains the visual design and responsive layout.
- `script.js` contains the calculator logic.
